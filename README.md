# Distance-Measuring-deivice
Basically this is a begineer friendly project for Electronics people who take interest in Home automations and Embedded Systems. This project focuses on digital distance measurement with accuracy upon the traditional measuring tapes. It uses an Ultrasonic Sensor HCSR04, an Arduino board and an LCD display. 
please install the Ardiuno IDE from here https://www.arduino.cc/en/software/
It's completely easy to setup the IDE, just follow the steps on the screen and you are ready to go. 
Connect your Arduino board with the given cable to the laptop to upload the code and give the power to the board.
In order to add the arduino board, just navigate to Tools -> Boards -> board manager -> install arduino AVR boards by Arduino.
After that it will automatically detect the board and you are good to go.
